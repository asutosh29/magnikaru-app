# Magnikaru
Simple Chess Server using flask, chessboard2.js and chess.js

# Setup Instructions
You 2 choices

## 1. Docker
```
> docker compose up --build
```

## 2. Manual Installation
## Environment
```
> virtualenv venv
> .\venv\Scripts\activate
```

## Install Dependancies
```
> .\venv\Scripts\activate
```

## Run the Server
```
> flask run
```
